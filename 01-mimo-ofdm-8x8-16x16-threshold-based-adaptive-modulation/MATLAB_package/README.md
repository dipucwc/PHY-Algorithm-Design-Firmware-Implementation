# MATLAB Package

This folder contains the complete MATLAB implementation of the **Scalable 8×8 and 16×16 MIMO-OFDM PHY Simulator with Threshold-Based Adaptive Modulation and Fixed-Rate Channel Coding**.

The package includes MIMO-OFDM transmitter and receiver processing, channel modeling, synchronization, LS and MMSE channel estimation, ZF and MMSE equalization, threshold-based modulation selection, fixed-rate convolutional coding, Viterbi decoding, and performance evaluation using BER, EVM, SINR, throughput, and channel-capacity metrics.

Run the main MATLAB script to configure the antenna size, execute the simulation, and generate the performance results and figures.
